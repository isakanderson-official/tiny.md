# About

This page lives at `site/content/about/index.md` and builds to `/about/`.

The mental model is intentionally small:

1. Folders are pages.
2. Each page uses an `index.md` file.
3. CSS lives in `site/theme/style.css`.
4. Build output goes to `dist/`.
